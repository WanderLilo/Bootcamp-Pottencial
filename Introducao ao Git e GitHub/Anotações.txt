Documentação Git/GitHub

Instalar Git

Criar novo repositório no GitHub
-New Repository, podendo informar o Nome Repositório, Descrição, se vai ser privado ou público, se adiciona ReadMe.MD, se adiciona .gitignore, tipo de licença.

Executar comandos do git
Selecionar a pasta, Git Bash Here

Clonar repositório existente
git clone [url do repositorio]

Verificar status
git status

Fazer modificação no repositório local
-Fazer alterações normalmente
-git add .
-git commit -m "Descricao do alteracao"

Subir alterações para repositório remoto
-git push origin main

Puchar alterações para repositório local
-git pull origin main
